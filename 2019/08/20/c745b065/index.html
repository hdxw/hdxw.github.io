更新于：2020年03月10日

最近参加了一次CTF比赛，签了个到，然后又对CTF感兴趣了。需要很多编码解码、进制转换之类的小工具，开始想用易语言写，后来发现不太适合，现在改用Python，这里顺便记录下搭建过程。

环境选择

版本选择

Pyhton的2.7.X版本和3.X版本有较大变动，不支持向后兼容，版本的选择有以下几个关注点：

  1. 是否兼容或可运行已有工具程序包和库
  2. 是否能够长久的使用（2.7.X版本2020年1月开始不再支持维护）
  3. 代码风格易学、易写程度

出于第二点，我最后选择使用最新版本的Python。

IDE选择

经过筛选10款最好的Python IDE、7款公认比较出色Python IDE，因为想用一个跨平台且轻量级的IDE，而visual studio code用来写博客了，不想一个编辑器多用，通过颜值最后选择了“Sublime Text”。
补充：Sublime Text中文乱码是真的难搞，最后改用Atom。

环境安装

1.下载安装

下载最新版本Python，选择“executable installer”。

下载Sublime Text，如果浏览器下载慢可以复制下载链接用迅雷下载，秒完成。

2.安装Package Control（插件管理）

Github地址：https://github.com/wbond/sublime_package_control

1.首先打开sublime，键盘按下ctrl + shift + p弹出界面，在对话框搜索Install Package Control，点击。点击完没有任何反应，得等一会儿，过了一会儿，才弹出提示界面，安装成功！

2.再次按下ctrl + shift + p，搜索Package Control: install Package，搜索出来后单击，然后就会再弹出一个对话框，搜索 ChineseLocalizations，点击然后sublime就自动变成中文了，可以在帮助-Language里面选择语言。

3.安装其他插件

点击菜单-首选项-Package Control，输入install并选中Install Package。在弹出的输入框搜索下面插件名，并点击安装。

1.Anaconda（最强的Python IDE插件）

2.SideBarEnhancements（侧边栏管理，打开文件夹会有效果）

3.HTML-CSS-JS Prettify（web环境使用，右键菜单中，需要配置nodejs路径）

4.测试运行

新建helloworld.py文件，输入print(‘Hello World!’)。

菜单-工具-编译系统-选中Anaconda Python Builder，回到代码ctrl + b编译运行
