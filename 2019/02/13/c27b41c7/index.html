打开Excel文件出现两个窗口

在安装Mysql后，每次打开Excel文档都会出现两个窗口，一个是要打开的文档，另一个是空白的Sheet1。

解决方案

第一步 打开配置项

打开Excel -> 文件 -> 选项 -> 加载项 -> 管理切换到“COM加载项” -> 转到

第二步 修改配置信息

取消勾选“MySQL For Excel”,点击确定，关闭已打开的Excel文档。重新打开Excel文档后就不会再出现空白的Sheet1了。

原因分析

“MySQL For Excel”是MySQL为Excel提供的一个数据库操作插件，可以方便的对数据进行导入导出扩展和编辑。开启后会在“数据”菜单下多出如下所示的选项，如有需要可以自行开启。
