简介

NoneBot是基于酷Q的python异步QQ机器人框架，本文记录了linux服务器下的搭建过程。

原理示意图

系统环境

腾讯云服务器1C1G1M
ubuntu
docker

步骤

安装酷Q

服务器安全组出站规则开放UDP:53（DNS）、TCP:80、TCP:433端口

入站规则添加TCP:9000端口（可指定本机IP），通过浏览器登录酷Q

在酷Q中重启CQHTTP插件(悬浮窗右键-应用-CQHTTP-重启应用)，然后就可以删除9000入站规则了

以上步骤完成后，内部端口信息如下：

5700 酷Q API 接口
8010 接收HTTP消息
8011 接收webSocket消息
9000 noVNC，酷Q web 端
